## What pianing is for

pianing provides a web-first workspace for piano practice with MIDI in mind. It is meant for players who want a more deliberate way to approach audition preparation and organize rehearsal time around MIDI-based work.

Rather than relying on a bulky local installation, the project keeps the experience centered in the browser. That makes it a practical fit for practice sessions, prep routines, and any situation where getting to the tools quickly matters more than managing a complex software stack.

## How to use it

Use pianing as a practice workspace for MIDI-based piano rehearsal.

Typical workflow:
1. Open the web app.
2. Load or connect the material you want to practice.
3. Work through the piece while focusing on audition preparation.
4. Repeat sections as needed during practice.
5. Adjust your routine based on the results of each session.

If you are hosting it locally, keep the HTML files available from the same folder so the browser can load the interface correctly.
